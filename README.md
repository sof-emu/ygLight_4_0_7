# ygLight 4.0.7

> 📦 คลังเก็บรักษา (Archive) ของ **ygLight 4.0.7** เพื่อการศึกษา ประวัติศาสตร์ซอฟต์แวร์ และการอนุรักษ์โปรแกรมเก่า

---

## เกี่ยวกับโครงการ

**ygLight** เป็นโปรแกรมที่เคยได้รับความนิยมในกลุ่มผู้เล่นเกม **Yulgang (โยวกัง)** ในช่วงประมาณปี **2008–2012**

ปัจจุบันเว็บไซต์และแหล่งดาวน์โหลดดั้งเดิมส่วนใหญ่ได้ปิดตัวลง ทำให้ไฟล์ของโปรแกรมแทบไม่หลงเหลืออยู่บนอินเทอร์เน็ตแล้ว

Repository นี้จัดทำขึ้นเพื่อ

* อนุรักษ์ซอฟต์แวร์เก่า (Software Preservation)
* ศึกษาประวัติของโปรแกรม
* ศึกษาการออกแบบ UI และ UX ของโปรแกรมในยุคนั้น
* ศึกษาโครงสร้างไฟล์และการจัดเก็บข้อมูล
* เป็นแหล่งอ้างอิงสำหรับนักสะสมซอฟต์แวร์และนักวิจัย

## ความปลอดภัย

เนื่องจากโปรแกรมมีอายุกว่า 15 ปี

จึงแนะนำให้

* ใช้งานภายใน Virtual Machine
* สแกนไฟล์ก่อนเปิดใช้งาน
* หลีกเลี่ยงการใช้งานบนเครื่องหลัก
* ไม่ใช้บัญชีผู้ใช้จริงกับโปรแกรม
* ตัดการเชื่อมต่ออินเทอร์เน็ตหากไม่มีความจำเป็น

## สถานะของโครงการ

โปรแกรมนี้เป็นซอฟต์แวร์รุ่นเก่า

* ไม่มีการพัฒนาเพิ่มเติม
* เว็บไซต์ดั้งเดิมปิดให้บริการแล้ว
* ลิงก์ดาวน์โหลดเดิมส่วนใหญ่สูญหาย
* Repository นี้มีวัตถุประสงค์เพื่อเก็บรักษาไฟล์ที่ยังหลงเหลืออยู่
